HoverZoomy – Image Zoom on Hover Privacy Policy

Effective date: March 22, 2026

HoverZoomy – Image Zoom on Hover runs locally in your browser and helps you preview larger versions of images while browsing.

Summary

HoverZoomy – Image Zoom on Hover does not collect, sell, or transmit personal data. The extension works locally and stores only user preferences.

Information the extension accesses

  • image elements and URLs on the current page
  • current page domain for site rules
  • user preferences

Information NOT collected

  • personal information
  • account or payment data
  • browsing history
  • analytics or tracking data
  • any external uploads

How data is stored

  • Preferences stored via chrome.storage.sync
  • Temporary data kept only in memory
  • No external servers used

Permissions

  • storage: for saving settings
  • site access: for detecting images and enabling hover zoom

Third-party services

No third-party services are used.

Changes

This policy will be updated if data usage changes.
