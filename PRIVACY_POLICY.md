# HoverZoomy – Image Zoom on Hover Privacy Policy

Effective date: March 22, 2026

HoverZoomy – Image Zoom on Hover runs locally in your browser and helps you preview larger versions of images while browsing.

## Summary

HoverZoomy – Image Zoom on Hover does not collect, sell, or transmit personal data to the developer. The extension works locally on the pages you open and stores only the settings needed to remember your preferences.

## Information the extension accesses

The extension may access:

- image elements, linked image URLs, and CSS background image URLs on the active page
- the current page domain to apply local allowlist or blocklist rules
- extension preferences saved by the user

## Information the extension does not collect

HoverZoomy – Image Zoom on Hover does not:

- collect personal information
- collect account credentials, payment data, or form input
- send browsing history to the developer
- use analytics, advertising trackers, or remote logging
- upload page content or images to any external server

## How data is stored

- User preferences are stored with `chrome.storage.sync`
- Temporary image cache data is kept only in memory and is cleared when the page is refreshed or closed
- No remote database or backend service is used

## Permissions

- `storage`: used to save settings such as language, preview size, site rules, and other preferences
- site access: required so the extension can detect images and show hover previews on pages the user visits

## Third-party services

HoverZoomy – Image Zoom on Hover does not use third-party analytics, ad networks, or remote APIs.

## Changes to this policy

If the extension's data behavior changes in the future, this policy should be updated before release.
