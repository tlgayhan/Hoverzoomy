# Privacy Policy for HoverZoomy – Image Zoom on Hover

Effective date: March 25, 2026

HoverZoomy – Image Zoom on Hover is a browser extension that displays larger image previews when users hover over supported images while browsing websites.

## Publisher

HoverZoomy Team  
Contact: ayhandogan4434@gmail.com

## Overview

HoverZoomy is designed to work locally in the browser. Its core purpose is to detect supported image sources on the page currently being viewed and display a larger preview for the user.

HoverZoomy does not collect, sell, rent, or otherwise transfer personal data to the publisher.

## Information Processed by the Extension

To provide its functionality, HoverZoomy may process the following information locally in the user’s browser:

- image URLs and image-related attributes such as `src`, `srcset`, and supported `data-*` values
- supported CSS background image URLs
- the current website hostname for site-specific allow and block rules
- extension settings selected by the user

This processing is used solely to provide hover-based image preview functionality and user-controlled extension behavior.

## Information Not Collected

HoverZoomy does not collect or transmit the following to the publisher:

- name, email address, phone number, or postal address
- passwords, authentication tokens, or payment details
- messages, form entries, or personal documents
- browsing history for analytics, profiling, or advertising
- personal content for resale, marketing, or unrelated tracking

## Network Activity

HoverZoomy does not send browsing data to the publisher’s servers.

In limited situations, the extension may request image or page resources directly from the same website the user is currently visiting in order to locate a higher-resolution image that is already available from that site. These requests are made locally by the browser as part of the extension’s visible functionality.

HoverZoomy does not use remote code, remote analytics, advertising SDKs, or tracking scripts.

## Local Storage

HoverZoomy stores only the settings necessary to remember user preferences. These settings may include:

- enabled or disabled state
- language preference
- preview delay and preview size settings
- dim background preference
- site allowlist and blocklist settings
- debug mode preference

Settings are stored using Chrome extension storage. Temporary runtime data, such as in-memory image caching, is kept only for performance purposes and is not used for profiling or analytics.

## Use of Data

Any information processed by HoverZoomy is used only to:

- detect supported images on the current page
- show larger image previews on hover
- attempt higher-resolution image sources when available
- apply user-selected settings and site rules

HoverZoomy does not use any data for:

- advertising
- analytics
- behavior profiling
- credit, lending, or eligibility decisions
- any purpose unrelated to the extension’s core functionality

## Data Sharing

HoverZoomy does not sell user data.

HoverZoomy does not share personal data with third parties, except where the browser directly requests content from the same site the user is actively visiting in order to display the requested image preview.

## Security

HoverZoomy is designed to operate with minimal permissions required for its functionality. It does not load or execute remotely hosted code.

## User Controls

Users may control the extension by:

- enabling or disabling it at any time
- allowing or blocking specific websites
- clearing extension settings
- uninstalling the extension at any time

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality or data practices change. The published version of this policy will reflect the most current terms.
